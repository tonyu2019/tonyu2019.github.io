iMac常用快捷键收藏

发表于 2020-12-16 分类于 os ， mac Changyan：
本文字数： 2.8k 阅读时长 ≈ 3 分钟

之前一直用的windows，换到iMac后虽然大家都说只是将control换成了command，其实仔细了解些，并只是这么简单，甚至在系统、应用快捷键里按键直接表示为图标，对于刚接触Mac用户来说，真是一脸懵逼，我还要查该符号对应的是哪个按键。

其他的也有很多差异的，比如重命名Mac中是Enter，显示桌面是fn+11，退出是command+q，当然都可以自己设定，其他差异大家慢慢摸索吧。本文基本收录了所有的快捷键及对应的图标，最后我想说对于开发人员来说mac确实要比windows友好的的多，当然抛除win开发人员和特别喜欢破解软件的人员(mac破解软件确实没有win丰富)。

全部快捷键

通用

command是Mac里最重要的修饰键，在大多数情况下相当于Windows下的Ctrl。

按键功能
fn + F11显示桌面
command(⌘) + Z撤销
command(⌘) + C复制
command(⌘) + V粘贴
command(⌘) + A全选
command(⌘) + S保存
command(⌘) + F查找
command(⌘) + H隐藏当前窗口(应用软件)
command(⌘) + option(⌥) + H隐藏所有非当前程序窗口
command(⌘) + M最小化窗口
command(⌘) + N新建
command(⌘) + O打开
command(⌘) + S保存
command(⌘) + shift(⇧) + S另存为
command(⌘) + P打印
command(⌘) + W关闭
command(⌘) + Q退出(按住不放大约1秒)

系统

按键功能
command(⌘) + shift(⇧) + eject(⏏)关闭显示器(程序正常运行)
command(⌘) + option(⌥) + esc(⎋)强制退出程序(类似window任务管理器)
command(⌘) + option(⌥) + eject(⏏)电脑睡眠
command(⌘) + shift(⇧) + Y用当前选中的文字生成便签

截图

按键功能
command(⌘) + shift(⇧) + 3截取全部屏幕到文件
command(⌘) + shift(⇧) + control(⌃) + 3截取全部屏幕到剪贴板
command(⌘) + shift(⇧) + 4截取所选屏幕区域到文件
command(⌘) + shift(⇧) + control(⌃) +4截取所选屏幕区域到剪贴板

文件管理器

按键功能
command(⌘) + shift(⇧) + N新建文件夹
command(⌘) + shift(⇧) + G调出【前往文件夹】窗口
return(↩︎)点击文件，按下重命名文件
command(⌘) + O选中文件/文件夹，打开所选项(文件夹/文件)
command(⌘) + option(⌥) + V剪切，其他位置文件/文件夹复制【command(⌘) + C】，目标位置按下该快捷键，被剪切到该位置
command(⌘) + ↑当前窗口打开上层文件夹(相当于window中的“向上”)
command(⌘) + delete(⌫)删除至废纸篓
command(⌘) + shift(⇧) + delete(⌫)清空废纸篓
space预览文件/文件夹
command(⌘) + I显示简介
command(⌘) + F搜索

切换

按键功能
command(⌘) + tab(⇥)应用程序间切换
command(⌘) + shift(⇧) + tab(⇥)应用程序间切换(反向)
command(⌘) + ~应用中的窗口间切换

浏览器

按键功能
command(⌘) + =放大
command(⌘) + -缩小
command(⌘) + L光标跳至地址栏
control(⌃) + tab(⇥)转下一个标签页
control(⌃) + shift(⇧) + tab(⇥)转上一个标签页
command(⌘) + T新建选项卡
command(⌘) + N新建浏览器窗口
command(⌘) + R重新加载当前标签页
command(⌘) + F搜索
command(⌘) + W关闭当前标签页

应用程序

按键功能
command(⌘) + H隐藏当前窗口
command(⌘) + option(⌥) + H隐藏所有非当前程序窗口
command(⌘) + Q退出最前面(当前)的程序
command(⌘) + Z撤销
command(⌘) + shift(⇧) + Z重做(撤销的逆向操作)
command(⌘) + tab(⇥)切换程序，停留在最近使用的程序上(windows中的alt+tab)
command(⌘) + option(⌥) + esc(⎋)打开【强制退出】窗口(类似window任务管理器)

文本处理

按键功能
command(⌘) + B切换所选文字粗体显示
fn + delete(⌫)向后删除
fn + ↑向上滚动一页（Page Up）
fn + ↓向下滚动一页（Page Down）
fn + ←滚动至文稿开头（Home）
fn + →滚动至文稿末尾（End）
command(⌘) + →将光标移至当前行的行尾
command(⌘) + ←将光标移至当前行的行首
command(⌘) + ↑将光标移至文稿末尾
command(⌘) + ↓将光标移至文稿开头
option(⌥) + →将光标移至下一个单词的末尾
option(⌥) + →将光标移至上一个单词的开头
control(⌃) + A移至行或段落的开头
